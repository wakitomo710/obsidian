```php
// Post モデル
public function tags()
{
    return $this->belongsToMany(
        Tag::class,         // 関連モデル
        'post_tag',         // 中間テーブル名
        'post_id',          // 中間テーブル側の「自分の外部キー」
        'tag_id',           // 中間テーブル側の「相手の外部キー」
        'id',               // 自分のローカルキー
        'id'                // 相手のローカルキー
    );
}
```
- 第1引数：関連モデル名（必須）
- 第2引数：中間テーブル名
- 第3引数：中間テーブル上の「自分の外部キー」
- 第4引数：中間テーブル上の「相手の外部キー」
- 第5引数：自分のテーブルのローカルキー
- 第6引数：相手テーブルのローカルキー
