#### 例（リファクタリング前）
```
$totalPrice = 0;
$discountedItems = [];

foreach ($items as $item){
	$totalPrice += $item->price;
	if ($item->isDiscounted) {
		$discountedItems[] = $item;
	}
}
```

↓

#### 例（リファクタリング後）
```
$totalPrice = array_sum(array_map(fn($item) => $item->price, $items)); $discountedItems = array_filter($items, fn($item) => $item->isDiscounted);
```
